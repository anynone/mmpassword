# Proposal: mmpassword MVP

| 字段 | 值 |
|------|-----|
| **变更名称** | mmpassword-mvp |
| **状态** | proposed |
| **创建日期** | 2026-03-26 |
| **技术栈** | Rust + Tauri 2.0 + React + TypeScript + Tailwind CSS |

---

## 概述

构建 mmpassword 密码管理器的最小可用产品 (MVP)，实现核心的密码库管理、加密存储和条目管理功能。

## 目标

1. **安全的本地密码存储** - 使用行业标准加密算法保护用户数据
2. **跨平台桌面应用** - 支持 Windows、macOS、Linux
3. **直观的用户界面** - 基于 Material Design 的现代化 UI
4. **完全离线可用** - 所有数据存储在本地，无需网络连接

## 范围

### 包含功能

| 模块 | 功能 |
|------|------|
| **密码库管理** | 创建、打开、解锁、锁定密码库 |
| **加密安全** | Argon2id KDF + ChaCha20-Poly1305 加密 |
| **条目管理** | CRUD 操作、复制密码、网站登录/安全笔记类型 |
| **分组管理** | 创建、重命名、删除分组（单级） |
| **UI 界面** | 欢迎页、解锁页、主界面、编辑页、新建密码库页 |
| **主题** | 深色/浅色主题切换 |

### 不包含（延后到 Phase 2/3）

- GitHub 仓库同步
- OTP/TOTP 支持
- 密码生成器
- 搜索/过滤功能
- 回收站
- 多级分组嵌套
- 附件管理
- 生物识别解锁
- 多密码库同时打开
- 系统托盘
- 自动更新

## 技术决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 前端框架 | React 18 | 生态成熟，组件库丰富 |
| 状态管理 | Zustand | 轻量级，API 简洁 |
| 加密算法 | ChaCha20-Poly1305 | 软件实现快，无硬件依赖 |
| 密钥派生 | Argon2id | 抗 GPU/ASIC 攻击，行业标准 |
| 文件格式 | .mmp (二进制加密) | 安全、紧凑 |

## 风险评估

| 风险 | 等级 | 缓解措施 |
|------|------|----------|
| 加密实现漏洞 | 高 | 使用成熟 crate (argon2, chacha20poly1305)，代码审查 |
| 密钥丢失无法恢复 | 高 | UI 明确提示用户备份主密码 |
| 跨平台兼容性 | 中 | 使用 Tauri 抽象层，测试三大平台 |
| 前端状态同步 | 中 | 使用 Zustand + Tauri 事件驱动 |

## 成功标准

- [ ] 用户可以创建新的加密密码库
- [ ] 用户可以通过主密码解锁密码库
- [ ] 用户可以添加/编辑/删除密码条目
- [ ] 密码库文件可以被安全地保存和加载
- [ ] 应用在 Windows/macOS/Linux 上正常运行
- [ ] 深色/浅色主题可以正常切换
