## 1. 页面结构总览

### 1.1 应用页面清单

| 页面名称 | 页面ID | 描述 | 入口 |
|---------|--------|------|------|
| 启动欢迎页 | P01 | 应用启动页面，显示最近打开的密码库 | 应用启动 |
| 解锁页面 | P02 | 输入主密码解锁密码库 | 点击密码库/自动锁定后 |
| 主界面 | P03 | 核心操作界面 | 解锁成功后 |
| 条目详情页 | P04 | 查看/编辑密码条目 | 点击条目 |
| 密码生成器 | P05 | 生成随机密码 | 工具栏/编辑条目时 |
| 设置页面 | P06 | 应用和密码库设置 | 菜单栏 |
| 导出页面 | P07 | 导出密码库数据 | 菜单栏 > 文件 |
| 新建密码库 | P08 | 创建新的密码库 | 启动页/菜单栏 |
| 关于页面 | P09 | 应用版本和版权信息 | 菜单栏 > 帮助 |
| GitHub连接配置 | P10 | 配置GitHub仓库连接 | 设置页面/启动页 |
| GitHub密码库管理 | P11 | 管理GitHub仓库中的密码库 | GitHub连接成功后 |

```
│   │                                                      │   │
│   │   ┌────────────────────────────────────────────┐    │   │
│   │   │  请输入主密码...                           │    │   │
│   │   └────────────────────────────────────────────┘    │   │
│   │                                       👁 显示密码    │   │
│   │                                                      │   │
│   │   ┌────────────────────────────────────────────┐    │   │
│   │   │              🔓 解锁密码库                  │    │   │
│   │   └────────────────────────────────────────────┘    │   │
│   │                                                      │   │
│   │   ─────────────── 或 ───────────────                │   │
│   │                                                      │   │
│   │   ┌────────────────────────────────────────────┐    │   │
│   │   │        🧬 使用 Touch ID 解锁                │    │   │
│   │   └────────────────────────────────────────────┘    │   │
│   │                                                      │   │
│   └──────────────────────────────────────────────────────┘   │
│                                                                │
│   ⚠️ 剩余尝试次数: 5/5                                         │
│                                                                │
│   ← 返回                                                       │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

#### 功能说明

| 元素 | 功能 | 交互 |
|------|------|------|
| 密码库名称 | 显示当前要解锁的密码库 | 无交互 |
| 密码输入框 | 输入主密码 | 支持显示/隐藏切换 |
| 解锁按钮 | 验证密码并解锁 | 错误提示 / 成功进入主界面 |
| 生物识别按钮 | Touch ID/Face ID解锁 | 仅macOS显示 |
| 尝试次数 | 显示剩余尝试机会 | 5次失败后锁定一段时间 |
| 返回按钮 | 返回启动页 | 点击 → P01 |

#### 状态说明

- **正常状态**: 显示密码输入框
- **错误状态**: 密码错误时显示红色边框和错误提示
- **锁定状态**: 多次失败后显示倒计时
- **生物识别可用**: macOS显示Touch ID按钮

| 仓库信息 | 显示当前连接的仓库和分支 | 刷新按钮更新文件列表 |
| 密码库列表 | 显示仓库中所有.mmp文件 | 勾选选择，双击打开 |
| 同步状态 | 显示待推送/待拉取的文件 | 点击展开详情 |
| 拉取远程 | 从GitHub拉取最新变更 | 显示拉取进度 |
| 推送本地 | 将本地变更推送到GitHub | 显示推送进度 |
| 新建密码库 | 在GitHub仓库中创建新密码库 | 弹出创建对话框 |
| 导入选中 | 导入选中的密码库文件 | 批量导入到本地 |

#### 状态说明

| 状态 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 已同步 | ✓ | 绿色 | 本地和远程一致 |
| 已导入 | ✓ | 绿色 | 文件已导入到本地 |
| 未导入 | ○ | 灰色 | 文件未导入 |
| 待推送 | ↑ | 蓝色 | 本地有未推送的更改 |
| 待拉取 | ↓ | 橙色 | 远程有新版本 |
| 冲突 | ⚠ | 红色 | 本地和远程都有更改 |
