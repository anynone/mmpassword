## 1. 产品概述

### 1.1 产品简介
mmpassword 是一款免费、开源、跨平台的**密码管理器**桌面应用程序。它采用强加密算法保护用户的密码和凭据数据，所有数据存储在加密的密码库文件(.mmp)中，完全存储在**本地文件系统**，用户对数据拥有完全控制权。

### 1.2 核心价值
- **安全性**: 所有敏感数据采用行业标准的强加密算法进行加密存储
- **完全本地**: 所有数据仅存储在本地，无云端依赖
- **跨平台**: 支持 Windows、macOS 和 Linux 多个操作系统
- **便携性**: 提供便携版本，支持自定义数据存储位置
- **开源**: 完全开源，代码透明可审计

### 1.3 目标用户
| 用户类型 | 描述 |
|---------|------|
| 个人用户 | 需要管理多个在线账户密码的技术用户 |
| 安全要求较高的用户 | 对数据安全有严格要求的个人或组织 |
| 敐藏敏感信息的用户 | 需要存储信用卡、SSH密钥等敏感信息的用户 |
| 离线环境用户 | 在无网络环境下使用密码管理的用户 |
| 数据主权用户 | 要求完全掌控自己数据的用户 |

## 2. 密码库存储

### 2.1 本地文件存储
- 密码库文件(.mmp)存储在本地文件系统
- 完全离线可用，无需网络连接
- 用户完全掌控数据存储位置
- 支持自定义存储路径

### 2.2 GitHub仓库存储
- 支持将密码库存储在GitHub仓库中
- 用户通过Personal Access Token进行身份认证
- 自动读取仓库指定分支中的.mmp密码库文件
- 保存后自动提交到GitHub仓库
- 支持多个GitHub仓库连接

### 2.3 数据迁移
- **导出功能**: 支持将密码库导出为CSV格式

---

## 3. 密码库管理

### 3.1 密码库加密
- 主密码用于加密整个密码库
- 采用行业标准的加密算法
- 支持两种密码库格式(Format A/Format B)

### 3.2 密码库状态
| 状态 | 描述 |
|------|------|
| 锁定 | 密码库已加密，需要主密码解锁 |
| 解锁 | 密码库已解密，可以正常使用 |

### 3.3 生物识别解锁
- 支持 Touch ID/Face ID 解锁（macOS）
- 可配置生物识别解锁次数限制
- 可配置生物识别超时时间
- 主密码作为备用解锁方式

### 3.4 多密码库支持
- 同时管理多个密码库
- 每个密码库独立的主密码
- 标签页式管理界面
- 密码库排序和重排

## 18. 与原方案差异说明

| 功能模块 | 原方案 | 调整后方案 |
|---------|--------|-----------|
| 密码库存储 | 本地 + 云存储(Dropbox/Google Drive/WebDAV) | **本地文件存储 + GitHub仓库** |
| 数据同步 | 支持云端同步 | GitHub仓库自动同步 |
| 离线缓存 | 需要缓存云端数据 | 本地+GitHub双存储 |
| 浏览器扩展集成 | 支持浏览器扩展访问 | 移除（依赖云连接） |
| 外部导入 | 支持从1Password/LastPass等导入 | **移除** |

**核心变化**: 产品定位为**本地优先密码管理器**，支持本地文件存储和GitHub仓库同步，用户完全掌控数据。
